1. Создается избранное место без указания обязательных параметров
Описание: можно создать избранное место, указав только параметр color, без указания параметров title, lat,lon.
Шаги:
1.Создать токен с помощью curl -X POST -s -D - https://regions-test.2gis.com/v1/auth/tokens 
2. Создать фикстуру, в которой указать только параметр цвета "null".
3. С помощью запроса (пример запроса: curl -X POST -s https://regions-test.2gis.com/v1/favorites) создать избранное место.
 Ожидаемый результат: Избранное место не создается. Ожидается ошибка 400.
 Фактический результат: Избранное место создается. Статус 200.
